The Calendar

At any time, you can navigate to your calendar by selecting the calendar icon in the top left corner of the page.

View Types

Switch the calendar view type by clicking one of the buttons in the top right corner of the page.

  • Week: Shows a detailed view of your events on a weekly basis. Click on the day header to open the Add Task Menu
  • Month: Shows an overall view of your events on a monthly basis
    • 1 click previews your day. This will show you what you have planned for the day. If you want more information, double-click the day to bring up the Add Task Menu
    • 2 clicks opens the Add Task Menu. This is a menu for you to add tasks and view the ones that already exist for the day so you can edit or delete them

Drag and Drop

You have the ability to drag and drop events to different days by clicking and holding on one of the dots (rectangles in week view) and releasing it on the new day that you want the event to occur on. Each event is connected to your To-Do List, whether you create, edit, or delete events. Click on the navigation menu icon in the top left corner and select To-Do List.

To-Do List

Adding Tasks

Add tasks by entering a name and a due date in the box at the bottom of the page. You can also add a type and description for extra details if you have those enabled in the quick settings

Editing Tasks

Click the pencil icon in the task card to begin editing. Perform your edits and click update to make the changes official. You can undo all changes by clicking cancel.

Completing Tasks

Drag a task card from the incomplete section and drop it into the completed section. In the To-Do List, you can only edit your tasks when they are incomplete. If you need to update information for a completed task in the To-Do List, you can drag it back from the Compmleted section to the Incomplete section.

Deleting Tasks

Click the delete icon in the task card.

Quick Add

Add tasks simply and quickly by providing a due date and a title. You can create multiple tasks at once before adding them all to your calendar. To add another task, click the plus icon on the right. Make sure to click Add to Calendar at the bottom right corner to officially create the tasks.

Side Navigation Menu

Select the three-line stack icon in the top left corner of the page to open the navigation menu.

Avatar

Edit your avatar picture by simply clicking it.

Quick Settings

Week View Default

Toggle this setting to Yes if you want to display the calendar as weeks whenever you navigate to it. Otherwise, the default view will be months. You can still change the calendar view by clicking the Week or Month buttons in the calendar.

Colors

Toggle this setting On if you want to be able to see colors reflected in your events. If it is Off, all events will be blue

  • Green: An event that is completed
  • Grey: An event that is in the past
  • Red: An event that is in the future, but very close to the current date
  • Yellow: An event that is in the future, but moderately close to the current date
  • Blue: An event that is in the future and not close to the current date

Task Type

Toggle this setting On to display the task type for each task. To edit task types, this setting must be On. You can edit the task type by opening the event in the Calendar or To-Do List.

Task Description

Toggle this setting On to display the task description for each task. To edit task descriptions, this setting must be On. You can edit the task description by opening the event in the Calendar or To-Do List.
